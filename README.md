#  Телефонная книга (Windows Forms)

## Описание
Классическое приложение для управления контактами с графическим интерфейсом. Реализованы все CRUD-операции, поиск, сортировка, сохранение и загрузка в бинарный файл.

## Технологии
- C# / .NET 6 (или ваша версия)
- Windows Forms
- Бинарная сериализация (BinaryWriter/Reader)

## Возможности
- Добавление, удаление, редактирование контактов (ФИО, телефон)
- Поиск по имени и номеру
- Автоматическая сортировка по имени (регистронезависимо)
- Сохранение и загрузка в .dat файл
- Интуитивно понятный интерфейс
